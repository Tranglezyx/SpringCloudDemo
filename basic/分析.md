# 分析总计

## 基本情况
往表中新增数据，然后使用jmeter进行压测，100个线程，每个线程执行1000次，总共10万条数据
采用springmvc、mybatis-plus，数据源使用hikari
```sql
create table sys_user
(
    id          bigint auto_increment,
    account     varchar(50)  null comment '账号',
    mobile      varchar(20)  null comment '电话',
    password    varchar(200) null comment '密码',
    nick_name   varchar(50)  null comment '别名',
    create_time bigint       null comment '创建时间',
    update_time bigint       null comment '更新时间',
    constraint sys_user_pk
        primary key (id)
);

```

## 接口同步新增
controller层直接往数据库insert
性能情况:1600TPS

## 异步新增
- 接入redis和kafka，redis进行幂等校验，kafka进行推送和异步消费入库
性能情况：8000QPS

- 接口不再进行redis幂等校验
  性能情况：10000QPS
